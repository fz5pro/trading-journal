Real-time sync across all devices — desktop, mobile, any browser. Powered by Firebase (free).

Setup Firebase — 5 minutes, free forever
1
Go to console.firebase.google.com → Add project → name it "FzTrade" → disable Analytics → Create
2
In the left sidebar → Build → Firestore Database → Create database → choose Start in test mode → pick any region → Done
3
Click the ⚙️ gear icon → Project settings → scroll to "Your apps" → click the </> Web icon → register app → copy the firebaseConfig values below
4
Open on any device/browser — same config = same data, auto-synced in real time ✓
